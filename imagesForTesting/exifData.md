# Exif Data from Test Files

Exif data for one image from each device type listed in the [deviceComparison](./deviceComparison/) folder was extracted using [ExifTool by Phil Harvey](https://exiftool.org/)

Looks like the first few lines of each report (File Name through Exif Byte Order, maybe?) are either added by ExifTool or from Windows, rather than the image file as it was on the device that took it.
